# Pakistan Income Tax Slab Analysis Tool

This application provides an interactive dashboard for analyzing income tax aggregates, collection efficiency, and the revenue impact of changing tax slab thresholds and rates in Pakistan.

## Features
- **Slab-wise Breakdown**: View filers and taxable income across current tax brackets.
- **Policy Simulation**: Adjust slab thresholds and tax rates to see projected revenue changes.
- **Efficiency Metrics**: Analyze tax collection effectiveness by bracket.

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `streamlit run tax_slab_simulation.py`

## Data Source
- `Slab wise Taxable Income Filers & Normal Tax_3012026.xlsx`

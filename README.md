# Pakistan Income Tax Analysis Suite

A unified repository containing tools for Personal Income Tax (PIT) analysis in Pakistan.

## Included Modules

### 1. PIT Microsimulation (`/pit_microsimulation`)
Interactive tool for simulating Personal Income Tax reforms and behavioral responses.

### 2. Slab Analysis Tool (`/slab_analysis`)
Dashboard for analyzing income tax aggregates, collection efficiency, and slab-wise breakdowns.

## Usage
Navigate to the specific module directory and refer to the instructions therein.
Most tools are built with Streamlit and require:
```bash
pip install -r requirements.txt
streamlit run <app_name>.py
```
